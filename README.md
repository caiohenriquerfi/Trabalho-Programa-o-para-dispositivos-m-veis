Este projeto foi desenvolvido utilizando React Native com uma arquitetura baseada em WebView, permitindo a integração fluida com o sistema web.
O aplicativo foi compilado através do EAS (Expo Application Services) para o ambiente Android. O arquivo binário (.apk) por ter mais de 25mb encontra-se disponível no google drive 
https://drive.google.com/drive/u/1/folders/1SFJh7e7XKO8A6z8PViZjtZ9OXXgyEITY

Nota: Por tratar-se de uma distribuição interna de desenvolvimento (.apk não assinado pela Google Play Store), alguns dispositivos Android podem exigir permissão manual para instalação de fontes desconhecidas ou apresentar alertas de segurança do sistema.

Para IOS e outros dispositiveis acessar pelo navegador https://contabilidade-motora-estacio.netlify.app/
